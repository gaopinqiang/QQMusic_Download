使用说明：

环境：电脑安装python2.7

脚本配置：
    1、在main.py中配置搜索的关键字（模拟在qq音乐首页搜索关键字），可以是歌名和歌手名。
        这是一个数组，可以添加多组。
    2、配置TOP_DOWNLOAD_NUMBER次数，这个参数意思就是下载前几个结果
        例如设置2，那么搜索到的结果页面就下载前2个歌曲。

运行：
    第一种方式：右击main.py,选择python.exe来运行
    第二种方式：如果配置环境变量，直接双击main.py
    第三种方式：打开cmd，python main.py即可运行。

结果：
    会在当前的目录下面创建一个Music文件夹，下载的结果会存在这个文件夹中。
